## 🚀 Quick Start: VS Code Setup

Follow these steps to get the environment running on your local machine.

### 1. Prerequisites
Ensure you have the following installed:
* **Python 3.10+**: [Download here](https://www.python.org/downloads/)
* **Git**: [Download here](https://git-scm.com/downloads)
* **VS Code**: [Download here](https://code.visualstudio.com/)

### 2. Clone the Repository
Open your terminal (or Command Prompt) and run:
```bash
git clone [https://github.com/YOUR_USERNAME/cyber-analytics-portfolio.git](https://github.com/YOUR_USERNAME/cyber-analytics-portfolio.git)
cd cyber-analytics-portfolio
