Project 01: Fake Website Classifier

The "So What?"
This tool calculates a "Randomness Score" for URLs. By flagging high-entropy domains, we can identify phishing infrastructure before it's used in a campaign, proactively blocking malicious traffic.

Protects the perimeter by stopping credential theft at the DNS level.

Project 02: Botnet Swarm Attribution

The "So What?"
Using the AbuseIPDB API and K-Means clustering, this script groups malicious IPs by confidence and volume. This allows us to distinguish between lone scanners and coordinated botnet swarms.

Enables automated, tier-based firewall responses based on attacker behavior clusters.
