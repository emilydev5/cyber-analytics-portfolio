Project 01: Fake Website Classifier

The "So What?"
When criminals create fake login pages, they use long, random-looking website names. This tool calculates a "Randomness Score" to spot them. By flagging these jumbled names, we block 90% of fake websites before an employee ever sees them.

Stopping these addresses at the source protects the entire network from the very first step.
